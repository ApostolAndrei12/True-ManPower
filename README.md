# True ManPower

Website-ul oficial True ManPower - Recrutare Internațională

## Despre Proiect

True ManPower este o companie specializată în recrutarea forței de muncă din țări non-UE pentru companiile din România. Website-ul oferă informații complete despre serviciile noastre, pachete personalizate și un proces simplu de contact.

## Caracteristici

- Design modern și responsive
- Suport multilingv (Română și Engleză)
- Pachete de servicii personalizate
- Formular de contact optimizat
- Secțiune dedicată industriilor
- Prezentare echipă cu slider interactiv

## Tehnologii Utilizate

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Radix UI
- Lucide Icons

## Instalare

1. Clonează repository-ul:
```bash
git clone https://github.com/your-username/True-ManPower.git
```

2. Instalează dependențele:
```bash
pnpm install
```

3. Rulează proiectul în modul development:
```bash
pnpm dev
```

4. Pentru build:
```bash
pnpm build
```

5. Pentru a rula versiunea build:
```bash
pnpm start
```
